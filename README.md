## Análise de Dados Operacionais sobre Horizonte Transportes

## Projeto 
- Com Excel, eu organizei um banco de dados no qual é possível cadastrar clientes e atualizar dados sempre que for necessário.
- Já no Power BI, verifiquei erros e editei com Power Query, criei medidas para serem usadas no dashboard, criei um dashboard para analisar a situação da empresa.

## Problemas Encontrados 
- Grande porcentual de atrasos (40%) 
- Poucos motoristas para uma quantidade alta de entregas 
- Pedidos de entregas muito distantes o que influencia nos atrasos 
- Muitas reclamações no atendimento devido a demora entre o pedido e a saída dele da empresa 
- Diminuição do lucro devido a essas problemas.

## Habilidades 
- Excel: macros, validação de dados, formatação das planilhas, criação de tabelas
- Power BI: dashborad, medidas, Power query

## Resultados e recomendações
- Contratação de pelo menos 5 motoristas 
- Organizar os motoristas por localização de forma permanente para acabar com atrasos 
- Melhorar o atendimento e comunicação com os clientes sendo mais ágil 
- Investir em premiação por resultados para ter funcionários mais eficientes e motivados
